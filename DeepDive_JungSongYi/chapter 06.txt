chapter 06

데이터타입
데이터 타입이란 값의 종류를 말한다
7개의 데이터타입은 원시타입과 객체타입으로 구분할 수 있다
변수선언 --> 컴파일러에게 식별자의 존재를 알림
변수정의 --> 변수에 값을 할당하여 변수의 실체를 명확히 함

    원시
    숫자, 문자열
    불리언(boolean) --> 논리적인 챁(true)/거짓(False)
    undefined --> var(let, const) 키워드에 암묵적으로 할당되는 값
    null --> 값이 없다는 것을 의도적으로 명시할때
    심벌(symbol) --> ES6에서 추가됨

    객체 --> 객체, 함수, 배열등

    6.1 숫자타입
    자바스크립트는 모든 수를 실수로 처리하며 정수를 표현하기 위한 데이터 타입은 존재하지 않는다
    2진수,8진수,16진수를 표현하기 위한 데이터타입을 제공하지 않기 때문에 모두 10진수로 해석된다
    모든 숫자를 실수로 처리하기때문에 정수로 표시하더라도 실수가 나올수 있다
        
        특별값
        infinity : 양의 무한대
        -infinity : 음의 무한대
        Nan : 산술 연산 불가(not -a-number)

    6.2 문자열 타입
    텍스트 데이터로 작은따옴표('') 큰따옴표("") 또는 백틱(`)으로 감싼다
    따옴표로 감싸는 이유 --> 키워드나 식별자같은 토큰을 구분하기 위해
    감싸지 않으면 --> 키워드나 식별자로 인식됨

    문자열을 감싸지 않는다면 공백문자를 포함시킬수 없다.
    문자열은 원시타입으로 변경이 불가능하다

    6.3 템플릿 리터럴
    편리한 문자열 처리기능을 제공 --> 런타임에 일반문자열로 변환되어 처리
    백틱(`)을 사용해 표현한다

        6.3.1 멀티라인 문자열
        일반 문자열에서 줄바꿈이 불가능 하지만 백슬래시로 시작하는 이스케이프 시퀀스를 사용시 가능하다
        줄바꿈 : /n
        var template = <ul>/n/t<li><a href "#">"home"</a><li>/n<ul>;
        백틱(`)사용
        var template = `<ul>/n/t<li><a href "#">"home"</a><li>/n<ul>`;

        6.3.2 표현식 삽입
        문자열 연산자 + 사용해 연결
        문자열 연산자 보다 가독성이 좋고 간편하게 조합할수 있다
        일반 문자열에 표현식 삽입 사용시 문자열로 취급된다

        var first = "red";
        var second = "cup";

        console.log(`My name is ${first} ${second}.`);

    6.4 불리언 타입
    논리적 참(true)/거짓(false)이 있다

    6.5 undefined 타입
    undefined값만 존재함

    6.6 null 타입
    대소문자 주의 해야된다 Null,NULL과 다르다
    변수의 값이 없음을 의도적으로 명시할때 사용한다
    변수에 null할당시 이전값을 참조하지 않겟다는 의미다 ---> 이전에 할당되있던 값을 제거하는 의미(가비지콜렉터 메모리 관리)
    함수가 유효한 값을 반환하지 못할시 null을 반환한다

    6.7 심벌타입
    변경 불가능하고 다른값고 중복 되지않는 유일한 값이다
    주로 이름이 충돌할 위험이 없는 프로퍼티키를 만들기 위해 사용한다
    symbol 함수를 호출해 생성한다

        var key = Symbol("key");
        console.log(typeof key);

        var obj = {};

        obj[key] = "value";
        console.log(obj[key]);

    6.8 객체타입
    자바스크립트는 객체기반 언어로 자바스크립를 이루고 있는것은 객체다

    6.9 데이터타입의 필요성
        값을 저장할때 확보해야하는 메모리 공간의 크기를 결정하기 위해
        값을 참조할때 한번에 읽어들여야 할 메모리 공간의 크기를 결정하기 위해
        메모리에서 읽어들인 2진수를 어떻게 해석할지 결정하기 위해

    6.10 동적 타이핑

        6.10.1 동적 타입 언어와 정적 타입 언어
        정적타입 : C나 C++ java ....
        변수를 선언할때 데이터 타입을 사전에 선언해야한다. 타입변경이 불가능하고 선언한 타입에 맞는값만 할당할수 있다
        타입체크 진행시 통과하지 못하면 에러를 발생시키고 프로그램 자체를 막는다 ----> 안정적 코드 구현을 위해 런타임에 발생하는 에러를 줄이는 장점이 있음

        동적타입 : 자바스크립트,파이썬,PHP ....
        자바스크립트는 어떠한 데이터의 값이라도 자유롭게 할당할수 있다
        typeof 연산자는 연산자뒤 피연산자의 데이터 타입을 문자열로 반환시킨다
        자바스크립트의 변수는 선언이 아닌 할당으로 타입이 결정된다 이는 재할당에 의해 변수가 언제든지 동적으로 변할수 있다 --> 동적 타이핑

        6.10.2 동적 타입 언어와 변수
        특징 : 편리함

        단점
        변수의 값이 언제든지 변경될수 있기때문에 추적이 어렵다 -> 값에 따라 타입도 변경될수 있다
        변수의 값을 확인하기전에는 타입을 확인할수 없다
        개발자의 의도와 상관없이 자바스크립트 엔진에 의해 암묵적으로 타입이 바뀔수 있다
        유연성 높지만 신뢰성 떨어짐
        
        변수사용시 주의사항(에러방지법)
        변수는 필요한 경우에만 제한적으로 사용가능하다
        변수의 유효범위(스코프)를 최대한 좁게 만든다
        변수보다는 상수를 사용해 값의 변경을 억제한다
        변수이름은 변수의 목적이나 의미를 파악할수 있도록 네이밍한다
