chapter 07

연산자
하나이상의 표현식을 대상으로 하나의 값을 만든다
연산의 대상 = 피연산자

    7.1 산술연산자
    피연산자를 대상으로 수학적 계산을 해 숫자값을 만든다

        7.1.1 이항 산술 연산자
        2개의 피연산자를 이용해 값을 만든다
        모든 이항산술 연산자는 피연산자의 값을 변경하는 부수적인 효과가 없다
        종류 +, -, *, /, %

        7.1.2 단항 산술 연산자
        1개의 피연산자를 이용해 값을 만든다
        종류 ++, --, +, -

        증가/감소 연산자 (++/--)는 피연산자의 값을 변경하는 부수효과가 있음

        위치에 따른 의미
        피연산자 앞 --> 먼저 값을 증가/감소 시킨뒤 수행
        피연산자 뒤 --> 먼저 수행뒤 값을 증가/감소

        var x = 5;

        선할당 후증가
        result = x++;
        console.log(result,x); 5 6

        선증가 후할당
        result = ++x;
        console.log(result,x); 7 7

        선할당 후감소
        result = x--;
        console.log(result,x); 7 6

        선감소 후할당
        result = --x;
        console.log(result,x); 5 5

        7.1.3 문자열 연결 연산자
        + 연산자는 피연산자중 하나이상이 문자열인 경우 문자열 연산자로 동작한다

    7.2 할당연산자
    우항에 있는 피연산자의 평가결과를 좌항에 있는 변수에 할당
    종류 : =, +=, -=, *=, /=, %=

    var x;
    console.log(x = 10);

    할당문도 값으로 평가 되므로 표현식이다

    7.3 비교연산자
    좌항과 우항의 피연산자를 비교 --> 결과를 불리언값으로 반환

        7.3.1 동등/일치 비교 연산자
        동등 비교 연산자 == 좌항과 우항의 피연산자를 비교할때 같은 값인지 확인
        일치 비교 연산자 === 좌항과 우항의 피연산자가 타입과 값이 같은지 확인
        NaN = 자신과 일치하지 않는 유일한 값 --> 조사시 빌트인 함수 Number.isNaN 사용

        부동등 비교 연산자 != 불일치 비교 연산자 !== --> 동등/일치 비교 연산자와 반대의 개념

        7.3.2 대소관계 비교 연산자
        피연산자의 크기를 비교하여 불리언값으로 반환한다
    
    7.4 삼항 조건 연산자
    조건식의 평가 결과에 따라서 반환할 값을 결정한다

    var result = score >= 60 ? 'pass' Fail';
    (score >= 60) === true
    (score >= 60) === False

    7.5 논리연산자
    논리 부정(!) 연산자는 언제나 불리언 값을 반환 피연산자가 불리언값이 아니면 불리언타입으로 암묵적 교체된다
    논리합(||)/논리곱(&&) 2개의 피연산자중 한쪽으로 평가됨

    7.6 쉼표연산자
    왼쪽 피연산자부터 차례로 평가하고 마지막 피연산자의 평가가 끝나면 결과를 반환한다.

    7.7 그룹연산자
    자신의 피연산자인 표현식을 제일 먼저 평가

    7.8 typeof 연산자
    피연산자의 데이터를 문자열로 반환한다
    typeof null 값을 확인하면 null이 아닌 object 를 반환 이는 버그다 null타입의 값을 확인할때는 일치연산자 === 를 써야한다

    var foo = null;
    typeof foo = null; ---> X
    foo === null ---> O

    7.9 지수연산자
    좌항의 피연산자를 밑으로이동
    우항의 피연산자를 지수로 거듭제곱해 숫자값을 만든다
    지수연산자는 이항 연산자 중에서 우선순위가 제일 높다

    7.11 연산자 부수효과
    다른 코드에 영향을 주는 효과
    할당 연산자(=), 증가/감소연산자(==/--) delete
